Class EventHub<Events>

A class that provides an event hub for managing custom events.

Type Parameters

  • Events extends AnyHubEvents = LSEvents

    The type of events that the EventHub will handle. This type should be an object with event names as keys and event data types as values. This make all events strongly typed.
